## Mau deploy?
```
Ambil session.json di termux & deploy seperti mendeploy rest api
Dan module harus lengkap biar bisa run otomatis

```

## CARA INSTALL DI TERMUX

```bash
> apt-get update
> apt-get upgrade
> npm install bash
> npm install git
> termux-setup-storage [Y]
> git clone https://github.com/Lexxy24/lordbot
> cd lordbot
> rm -r session.json
> bash install.sh 
> npm start
> Now scan the QR
```
